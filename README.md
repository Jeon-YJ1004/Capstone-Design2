## 3D Solar System

서울과학기술대학교 졸업작품   

학교 교과서에 나오는 태양계나 흔히 떠올릴 수 있는 행성들의 크기는 종이에 담길 수 있게 
또는 이해하기 쉽게하기 위하여 실제 태양과 행성들의 차이를 줄여 나타냈다.   
목표는 사용자로 하여금 실제 비율을 느낄 수 있게 하고, 단순한 물체의 크기 차이가 아니라 
애니메이션을 통해 동적으로 느낄 수 있게 하는 것이다. 또한 행성들의 상세 정보도 알 수 있다.
